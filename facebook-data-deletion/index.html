Procédure conforme (Facebook / Instagram)

Comment demander la suppression de vos données ?

Si vous avez utilisé la connexion ou les fonctionnalités Facebook / Instagram avec notre application et souhaitez supprimer vos données, suivez ces étapes :

  1. Retirer l’application depuis Facebook
    Paramètres → Applications et sites web → Actifs → « Nexyna AI » → Supprimer.
  2. Nous adresser une demande
    Écrivez à amine@nexyna.ai (e-mail + toute info utile, ex. ID utilisateur Facebook).
  3. Délai
    Suppression/anonymisation sous 30 jours. Confirmation envoyée par e-mail.

Portée

Concerne les données nécessaires au service (ex. jetons d’accès, identifiants techniques, journaux). Données légales minimisées et supprimées dès que possible.
